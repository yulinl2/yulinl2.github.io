Paper-writing guide

Walk-throughs for the manuscript author finishing the FaiREE-with-missing-data paper and getting it to submission. Sister sections (tutorials, audits) cover the code side; this one covers the prose side.

Audience tier: far (paper author + reviewer-anticipation). Assumes the recovery branch artifacts (26 .tex drafts + 32 result dirs + closed W-1/W-2 + 93 board items) and walks them to a single PDF.

Guide 1 · 7 min
Section map & integration order
What each .tex in _drafts/ contains and what goes where in the final paper. The earlier 5 residual forward-references (prop:pessimistic, sec:discussion, sec:dr-ablation, sec:shift-ablation, sec:shrinkage_validation) are now all resolved.
Guide 2 · 5 min
Compile checklist
outline.tex → single PDF: required packages, common LaTeX errors and fixes, how to swap the in-repo main.tex stub, the short-vs-long flag pattern.
Guide 3 · 10 min
Submission readiness checklist
Pre-submission gates: dangling-refs resolved, figures finalized, supplement packaged, DEC-2/DEC-3 prose decisions, citation hygiene, anonymization. Plus a venue-by-venue prep notes (NeurIPS / ICML / TMLR / JMLR).
Guide 4 · 8 min
Reproducibility supplement
What to package for the camera-ready supplement: code freeze, result-dir snapshot, requirements pin, headline-figure regeneration scripts, README for reviewers.
