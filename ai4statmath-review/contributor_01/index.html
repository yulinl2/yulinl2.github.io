# Solution

**Step 1: Find the distance traveled**

Since all 3 kangaroos travel at the same speed and together take 18 hours, each individual kangaroo takes 18 hours to cross the highway.

Let's say each kangaroo travels at speed $v$.
- Distance = $v \times 18$ hours

**Step 2: Find the speed of each turtle**

Each turtle travels at half the speed of a kangaroo:
- Turtle speed = $\frac{v}{2}$

**Step 3: Calculate time for turtles**

Since all 4 turtles travel at the same speed, each turtle takes the same time to cross.

Using distance = speed × time:
$$18v = \frac{v}{2} \times t$$

$$t = \frac{18v}{\frac{v}{2}} = 18v \times \frac{2}{v} = 36 \text{ hours}$$

The number of turtles doesn't affect the time—only their speed matters.

**Final answer: 36 hours**
