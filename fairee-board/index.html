How this board works
  • Edit any item: click a card. Change status / owner / priority / detail; close the popup to commit it in-memory.
  • Save: paste a fine-grained PAT once (scoped to yulinl2/yulinl2.github.io, Contents: Read & write). The button commits fairee-board/assets/data/roadmap.json back into the repo on the chosen branch.
  • Autosave: every change is mirrored to browser localStorage so a refresh doesn't lose work.
  • Export = download JSON snapshot. Import = paste a JSON to replace state (rarely needed).
