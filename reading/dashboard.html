How the batch download works

Primary path — “Download ZIP”: your browser fetches the selected PDFs and bundles them into a flat bibliography-pdfs.zip where each file is renamed <category-prefix>__<slug>.pdf (e.g. inference-on-inference__zhang-2024-winners-with-confidence.pdf). arXiv usually allows cross-origin fetches; if a particular host blocks CORS you'll see a per-file error in the status bar.

Fallback — “Script (.sh)”: downloads a bash script that uses curl to fetch all selected PDFs into a single flat folder with the same category-prefixed filenames. Run it locally when the browser path can't reach a host.
